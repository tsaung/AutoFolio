# AI-Powered Portfolio (RAG)

An open-source, AI-powered portfolio for professionals. This application serves as a **Personal RAG (Retrieval-Augmented Generation) App**, allowing visitors (recruiters, clients, HR) to ask questions and receive answers based on the professional's context, resume, and work history.
