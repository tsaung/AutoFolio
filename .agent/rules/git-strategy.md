# Git Strategy

To maintain a clean and conflict-free history in this Autonomous AI Environment:

1. **Manual Merging:** Do not run automated sync scripts. Merging `development` into your branch is a manual task performed by the Human Lead if necessary. Agents should focus on their specific task without syncing `development` automatically.
2. **Merge Strategy:** If you must merge, use `git merge` (not rebase) to incorporate changes.
   - Focus on your specific task. Let the Human Lead handle complex integrations.
