# Temporary Files & Scratchpad

- **Scratchpad Location:** If you need to create temporary files (logs, data validation scripts, scratchpad code), **ALWAYS** use the `.ai/temp/` directory.
- **Git Hygiene:** Do NOT commit files in `.ai/temp/` or any `*.tmp` files.
